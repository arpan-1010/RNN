# 📚 Simple RNN for Question-Answering

This repository contains a simple implementation of a Recurrent Neural Network (RNN) using PyTorch for a basic Question-Answering (QA) task. The model is trained on a custom dataset of question-answer pairs and learns to predict answers based on input questions.

---

## 🚀 Features
- 📄 Loads a custom QA dataset from a CSV file.
- 🔤 Text preprocessing with tokenization and vocabulary creation.
- 🔢 Converts text into numerical indices for model training.
- 🧠 Implements a simple RNN with an embedding layer.
- 🏋️ Training loop with CrossEntropy loss and Adam optimizer.
- 🔮 Prediction function for answering new questions.

---

## 📂 Dataset Used : 
- Custom question answering dataset
